## 这个文件夹总有4个文件
## 第一个是embedding，负责将自然语言计算成向量
## 第二个是matching，将概念的向量和评论的向量进行匹配
## 第三个optimization，负责优化其中的阈值
## 第四个是前端，我使用streamlit简单制作了一个可视化前端
